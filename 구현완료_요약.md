## 🚀 API 엔드포인트

### 인증
- POST /api/auth/register - 회원가입
- POST /api/auth/login - 로그인

### 여행 계획
- POST /api/plans - 계획 생성
- GET /api/plans - 사용자 계획 목록
- GET /api/plans/{id} - 계획 상세
- DELETE /api/plans/{id} - 계획 삭제
- DELETE /api/plans/destinations/{id} - 관광지 삭제
- DELETE /api/plans/accommodations/{id} - 숙소 삭제
- POST /api/plans/{id}/ai-recommendations - AI 추가 추천

### 여행 기록
- POST /api/records - 기록 작성
- GET /api/records/plan/{planId} - 계획별 기록
- GET /api/records/{id} - 기록 상세
- PUT /api/records/{id} - 기록 수정
- DELETE /api/records/{id} - 기록 삭제

### 게시판
- GET /api/board/posts - 공개 게시글 목록
- GET /api/board/posts/{id} - 게시글 상세
- POST /api/board/posts - 게시글 작성
- POST /api/board/posts/{id}/like - 좋아요
- POST /api/board/posts/{id}/comments - 댓글 작성
- GET /api/board/posts/{id}/comments - 댓글 목록

### PDF 내보내기
- GET /api/export/pdf/{planId} - PDF 데이터
